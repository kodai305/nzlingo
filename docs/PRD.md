# NZLingo - PRD (Product Requirements Document)

## 1. Overview

NZLingoは、海外旅行や日常で英語を話せるようになりたい人向けの、1日5分の英会話練習Webアプリ。映画やドラマの実際のセリフを題材に、ネイティブらしい自然な英語表現を楽しく学べる。

## 2. Background / Problem

- 既存の英語学習アプリ（Duolingo等）は教科書的な表現が多く、実際のネイティブ表現との乖離がある
- 子育て中など忙しいユーザーにとって、まとまった学習時間の確保が難しい
- 「1日5分、1フレーズ」という極限のシンプルさで、始めるハードルを最小化したい

## 3. Target User

- 主要ユーザー: 30代女性、子育て中で時間が限られているが、旅行等で英語を話せるようになりたい
- 英語レベル: 初中級（中学〜高校英語は一通り学んだが会話に自信がない）
- 利用環境: iPhone（Safari）からPWAとして利用

## 4. Core Concept

> 毎日1つ、映画・ドラマの名セリフを聞いて、理解して、口に出す。

## 5. Features (MVP)

### 5.1 Daily Phrase（1日1フレーズ）

- 映画・ドラマの実際のセリフ（ワンフレーズ）を毎日1つ表示
- 以下の情報を含む:
  - **フレーズ（英語）**: セリフ本文
  - **日本語訳**: 自然な日本語訳
  - **出典**: 映画/ドラマ名、キャラクター名
  - **映画ジャンル**: アクション、ロマンス、コメディ等
  - **シーンイラスト**: どういう場面かがわかるAI生成イラスト
  - **音声再生**: フレーズのネイティブ発音を聞ける（TTS）
  - **解説**: 単語の意味、言い回しのニュアンス、どういう場面で使えるか、文法的なポイント

### 5.2 Authentication

- Google認証（OAuth 2.0）
- メールアドレス認証（マジックリンクまたはパスワード）
- インターネット公開のため認証必須

### 5.3 Progress Tracking（学習記録）

- 学習した日を記録（カレンダーまたはストリーク表示）
- 過去に学んだフレーズの一覧・復習

### 5.4 PWA

- ホーム画面追加でアプリライクな体験
- オフライン対応（キャッシュ済みコンテンツの閲覧）
- プッシュ通知（1日1回のリマインダー、任意設定）

## 6. Features (Future / Post-MVP)

- 発音判定（Web Speech API → ユーザーの発音を認識・評価）
- ジャンル別フィルター（好きなジャンルのフレーズだけ出す）
- お気に入り登録
- クイズ形式の復習（穴埋め、リスニングクイズ等）
- AIによるコンテンツ動的生成
- SNSシェア（今日のフレーズをシェア）

## 7. Content Strategy

### 初期コンテンツ（MVP）

- AIを使って事前に100〜200フレーズ分のデータを一括生成
- 各フレーズにAI生成イラスト（DALL-E等）を事前作成
- JSON/DBに格納し、固定コンテンツとして配信
- 映画ジャンルのバランスを考慮して選定

### コンテンツ構造（1フレーズあたり）

```json
{
  "id": "phrase_001",
  "phrase_en": "You had me at hello.",
  "phrase_ja": "「ハロー」で私の心は決まってた。",
  "source": {
    "title": "Jerry Maguire",
    "title_ja": "ザ・エージェント",
    "character": "Dorothy Boyd",
    "year": 1996
  },
  "genre": "romance",
  "image_url": "/images/phrases/phrase_001.webp",
  "audio_url": "/audio/phrases/phrase_001.mp3",
  "explanation": {
    "summary": "映画のクライマックスで使われる有名なセリフ。「had me」は「心を掴んだ」という意味。",
    "vocabulary": [
      {
        "word": "had me",
        "meaning": "（心を）掴んだ、虜にした",
        "note": "have + 人 で「〜を虜にする」というカジュアルな表現"
      }
    ],
    "usage": "相手に「最初から好きだった」と伝えたい時に使える表現。告白やロマンチックな場面で。",
    "grammar": "「had me at ~」で「〜の時点で心を掴まれた」というイディオム。atの後には具体的な瞬間やきっかけが入る。"
  },
  "difficulty": "beginner"
}
```

### コンテンツ選定基準

ターゲットユーザー（30代日本人女性）がある程度知っていそうな作品を優先的に選定する。

**映画**
- 日本でヒットした洋画（興行収入上位、地上波放送された作品等）
- 例: ハリー・ポッター、アナと雪の女王、タイタニック、プラダを着た悪魔、ラ・ラ・ランド、ショーシャンクの空に、ホーム・アローン、バック・トゥ・ザ・フューチャー、マイ・インターン等

**海外ドラマ**
- 日本でも知名度の高い作品
- 例: フレンズ、ゴシップガール、セックス・アンド・ザ・シティ、ストレンジャー・シングス、エミリー、パリへ行く等

**アニメ（英語吹替版）**
- 日本発のアニメの英語版セリフ — 元のアニメを知っているので親しみやすい
- 例: ジブリ作品（千と千尋、もののけ姫等）、鬼滅の刃、SPY×FAMILY、進撃の巨人等
- ディズニー/ピクサー作品も含む

**選定NG基準**
- 日本での知名度が低い作品
- 過度に暴力的・下品なセリフ
- 専門用語が多すぎて実用性がないもの（法廷ドラマの専門用語等）

### 著作権対応

- 映画・ドラマの短いフレーズ（1〜2文）を引用として使用
- 出典（作品名、キャラクター名、公開年）を必ず明記
- 商用利用ではなく個人/家族利用が主目的

## 8. Tech Stack

| Layer | Technology | Reason |
|-------|-----------|--------|
| Frontend | Next.js (App Router) | SSR/SSG対応、PWA化しやすい |
| Styling | Tailwind CSS | 高速開発、レスポンシブ対応 |
| Auth | Supabase Auth (Google OAuth + マジックリンク) | DB+認証を一元化。`allowed_users`でホワイトリスト制御 |
| Database | Supabase (PostgreSQL) | 無料枠あり、RLSでセキュリティ担保 |
| Storage | Supabase Storage | 音声・画像をDB+一元管理 |
| TTS (音声) | Amazon Polly (事前生成) + Web Speech API (フォールバック) | 高品質な事前生成音声、未生成時はブラウザTTSで代替 |
| Image | Amazon Bedrock Titan Image Generator v2 (事前生成) | AWS内完結、コスト低い |
| Content | Amazon Bedrock Claude Sonnet 4.6 | フレーズデータの一括生成 |
| Hosting | Vercel (MVP) | Next.jsとの親和性。将来はAmplifyへ移行検討 |
| PWA | Serwist | next-pwaはメンテ停止。Serwistが後継 |

## 9. User Flow (MVP)

```
1. 初回アクセス → ログイン画面（Google or Email）
2. ログイン後 → 今日のフレーズ画面
   ├── シーンイラストが表示される
   ├── フレーズ（英語）が表示される
   ├── 再生ボタンを押す → ネイティブ発音が流れる
   ├── 日本語訳を確認
   ├── 解説を読む（単語・言い回し・使える場面）
   └── 「学習完了」ボタンを押す
3. 学習完了 → 完了アニメーション + 明日もがんばろうメッセージ
4. 過去のフレーズ → 履歴一覧から復習可能
```

## 10. Design Principles

- **とにかくシンプル**: 1画面で完結。迷う要素を排除
- **5分以内**: 開いて、聞いて、読んで、閉じる。それだけ
- **楽しさ優先**: 映画のシーンを想像しながら学ぶワクワク感
- **プレッシャーゼロ**: ストリークが途切れても罰なし。いつでも再開できる

## 11. Success Metrics

- 週3回以上アクセスするか（継続率）
- 1セッション5分以内で完了するか（手軽さ）
- 「楽しい」「続けられそう」と感じるか（定性フィードバック）

## 12. Scope / Out of Scope

### In Scope (MVP)
- 1日1フレーズの表示・音声・解説
- Google/Email認証
- 学習履歴の記録
- PWA対応
- モバイルファーストUI

### Out of Scope (MVP)
- 発音判定
- ジャンルフィルター
- クイズ/ゲーミフィケーション
- 課金機能
- 複数言語対応
